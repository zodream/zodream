## 配置

```PHP
    'mail' => array(
        'host'     => 'smtp.qq.com', // 邮件服务器地址
        'port'     => 25,            // 服务器端口
        'user'     => '1@qq.com',    // 账号
        'email'    => '',            // 发送者地址，可空
        'name'     => 'ZoDream',     // 发送者名称
        'password' => '1'            // 密码
    ),
```
